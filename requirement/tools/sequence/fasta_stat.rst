
工具说明
==========================
fasta文件序列信息统计

Path
-----------

**sequence.fasta_stat**

程序安装路径
-----------------------------------

/mnt/ilustre/users/sanger/app

功能和用途描述
-----------------------------------

通过运行seqstat软件，统计fasta文件的序列总数目、最长最短序列、每条序列的长度等信息


使用程序
-----------------------------------

seqstat

主要命令及功能模块
-----------------------------------

seqstat.py -i fasta

参数设计
-----------------------------------

::

    {"name": "fasta", "type": "infile", "format": "sequence.fasta"} # 输入文件

运行逻辑
-----------------------------------

传入fasta文件后运行seqstat软件统计Fasta文件的序列信息
