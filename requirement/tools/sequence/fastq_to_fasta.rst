工具说明
==========================

Path
-----------

**sequence.fastq_to_fasta**

功能和用途描述
-----------------------------------

将fastq文件转换成fasta文件

主要命令及功能模块
-----------------------------------




参数设计
-----------------------------------

::

            {"name": "fastq_input", "type": "infile", "format": "fastq"},  # 输入文件
            {"name": "fasta_id", "type": "string", "default": "none"}  # 自定义序列id，默认将保留fastq文件id，将'@'改成'>'

运行逻辑
-----------------------------------
当传入参数fastq_input,就可以运行此模块,fasta_id为可选参数
