格式说明
==========================

Path
-----------

**seq.fastq_dir**


功能和用途描述
-----------------------------------

用于存储以某种规则放置在一起fastq文件


格式定义文档
-----------------------------------

pass

格式举例
-----------------------------------

::
 ./
 ../
 fastq1.fastq
 fastq2.fq
 fastq3.fastq.gz
 fastq4.fq.gz



属性及其含义
-----------------------------------

* ``file_number``   文件夹中fastq文件的数目

相关方法
-----------------------------------

``get_fastq_number``    获取这个文件夹下fastq文件的数目
``convert_to_fasta``    将这个文件夹转化为fasta_dir, 里面包含的fastq文件都将一一转化
